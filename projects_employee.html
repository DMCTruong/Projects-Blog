Employee Management System

Purpose

The Employee Management System is a project that was created in class. It is a basic system that helps managers manage employees. The system has the functions listed below.

  • Registration system
  • Login system
  • Search for employee information
  • Return number of employee registered in the system
  • Print all employee information to the screen
