Employee Management System

Purpose

The Employee Management System is a project that was created in class. It is a basic system that helps managers manage employees. The system has the functions listed below.

  • Registration system
  • Login system
  • Search for employee information
  • Return number of employee registered in the system
  • Print all employee information to the screen

Installation

To install the program:

  1. Make sure that you have Java installed.
  2. Download the program from GitHub.
  3. Extract the files into a folder of your choosing.
  4. Compile the program using an IDE or a command line

Be sure to check out the instructions in the README if you would like to download.

Usage

To use the program:

  1. Run the program
  2. Begin by registering
  3. Next, log into the program using the information that was registered
  4. If log in was a success, then one can have access to the rest of the program and its features such as database search and printing database information.
