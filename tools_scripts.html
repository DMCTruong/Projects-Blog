Github Bash Scripts

This repository contains some shell scripts that runs common Github commands such as "git init" and "git commit" using Git Bash. The purpose of these shell scripts is to help speed up the development process of future projects, simplify, and automate repetitive tasks.

Shell Scripts and Descriptions

  • setup_git.sh: Sets up a new repository and requests the user for information and the repository origin.
  • clone_git.sh: Clones the GitHub repository.
  • set_tracking_branch.sh: Sets or switch the current branch tracking information.
  • commit_and_push_git.sh: Adds and commits files to the web repository.
  • remove_deleted_commit_files_git.sh: Removes deleted commit files.
  • rollback_a_commit_git.sh: Rolls back a commit.
  • generate_sshkey.sh: Generates a ssh-key and copies it to the clipboard.

Installation

  • Download the .sh files from GitHub
  • Move it to the root of the repository

Usage:

  • Double click to run it. Some of the scripts will ask the user for input.
