Socket Stream Server and Client Implementation

The Socket Stream Server and Client Implementation was a small project created for class to study about the inner workings of servers and their clients. The server can take login information from the client and if the client is successfully logged in, the client can send a message to the server from which the server displays the received message to the screen. The project will be expanded further in the future with added functionality.

Installation

To install the program:

  1. Download the program at GitHub
  2. Extract the program to a folder of your choosing and make sure the text file is in the same folder as the program.
  3. Make sure "gcc" is installed
  4. Compile the two files.
      5. - Compile server.c with: gcc -o server server.c
      - Compile client.c with: gcc -o client client.c
  5. Once the two files are compiled, proceed to the Usage section for information on how to run it.

Be sure to check out the instructions in the README in the root project directory if you would like to download.

Usage

To use the program:

  1. Run the compiled server program first on one terminal, then leave the server program as it is.
  2. Next, run the compiled client program on a separate terminal
  3. On the client terminal, enter one of the user name and password pairs that is included in the passwd.txt. Then, wait for the server to verify and accept it.
  4. If the server accepts the user name and password, the user may enter a message and send it to the server. If all goes well, the server should send the message right back and that would be the end of the program.
  5. If the server does not accept the user name and password, then the user must re-enter the username and password. Be sure that the user name and password are one of the few that is already listed in the passwd.txt.
